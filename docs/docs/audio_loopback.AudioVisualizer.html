audio_loopback.AudioVisualizer¶

class audio_loopback.AudioVisualizer(device_controller: audio_loopback.audio_visualizer.DeviceController, audio_controller: Optional[audio_loopback.audio_loopback.AudioController] = None, fade: float = 0.8, delay: float = 0.01, dampen: int = 2200, ceiling: int = 2850, ambient_brightness_coef: float = 0.1)¶

Audio visualizer class, virtual parent class. Avoid. convert audio data to Keyboard color_matrix

Parameters

  • device_controller – Device controller class responsible of audio to color translation.

  • audio_controller – Audio controller object to read audio data from.

  • fade – Fade speed for the visualization

  • delay – Step delay amount between animations

  • dampen – Dampening coefficient for visualization

  • ceiling – Ceiling value for the visualization. Max value expected from the audio controller DFFT.

  • ambient_brightness_coef – Background brightness behind the visualization
