audio_loopback.AudioController¶

class audio_loopback.AudioController(aud_format: int = 8, channels: int = 1, rate: int = 48000, device_name: str = 'pulse', dampen: float = - 25)¶

Simple class to capture fft data from audio stream

Parameters

  • aud_format – Audio output format for PyAudio. Default is pyaudio.paInt16

  • channels – The desired number of input channels for PyAudio. Default is 1

  • rate – Sampling Rate. 48000 by default. Lower sampling rates will take longer to collect.

  • device_name – Name of the audio device to attach to.

  • dampen – Level of audio dampening to apply. Ideally, silence should yield [0,0,0,0…] and the slightest noise should yield non-zero.
