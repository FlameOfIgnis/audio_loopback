class audio_loopback.audio_loopback.AudioController(aud_format: int = 8, channels: int = 1, rate: int = 48000, device_name: str = 'pulse', dampen: float = - 25)¶

Bases: object

Simple class to capture fft data from audio stream

Parameters

  • aud_format – Audio output format for PyAudio. Default is pyaudio.paInt16

  • channels – The desired number of input channels for PyAudio. Default is 1

  • rate – Sampling Rate. 48000 by default. Lower sampling rates will take longer to collect.

  • device_name – Name of the audio device to attach to.

  • dampen – Level of audio dampening to apply. Ideally, silence should yield [0,0,0,0…] and the slightest noise should yield non-zero.

static dfft_reduce(dfft: list, count: int = 25, reduction_coef: float = 50, top: int = 100, skip: int = 30) → list¶

Reduce an fft result to “count” elements. This is done by summing “reduction_coef” many results into a single value. Whatever remains is ignored after count*reduction_coef elements.

Parameters

  • dfft – DFFT data array to reduce

  • count – Number of partitions to split the spectrum into. Number of steps on the graph from 0hz to max hz. Default is 25.

  • reduction_coef – Coefficient to to scale the output with. Default is 50. Sensitivity.

  • top – Maximum value the graph will be scaled up to. Default is 100.

  • skip – Size of partitions to skip between each partition. Larger is faster, but less precise.

Return type

list

Returns

A list of audio output after fast fourier transform
